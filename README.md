# 📔 감정 일기장

> 일기와 감정을 함께 기록하는 웹 애플리케이션

감정 일기장은 일상의 기록과 함께 당신의 감정을 함께 기록할 수 있는 서비스입니다.
React와 React Router를 활용한 SPA(Single Page Application)로 구현되어 부드러운 사용자 경험을 제공합니다.

---

## ✨ 주요 기능

- 📝 **일기 작성**: 새로운 일기를 작성하고 저장할 수 있습니다
- 😊 **감정 기록**: 일기와 함께 당시의 감정을 함께 기록할 수 있습니다
- 📋 **일기 목록**: 작성한 모든 일기를 한눈에 볼 수 있습니다
- 🔍 **일기 상세**: 저장된 일기를 상세히 조회할 수 있습니다

---

## 🛠️ 기술 스택

- React + JavaScript

## 📁 프로젝트 구조

```
emotionDiary/
├── src/
│   ├── pages/
│   │   ├── Home.jsx      # 메인 페이지 (일기 목록)
│   │   ├── New.jsx       # 일기 작성 페이지
│   │   ├── Diary.jsx     # 일기 상세 페이지
│   │   └── Notfound.jsx  # 404 페이지
│   ├── App.jsx           # 라우팅 설정
│   ├── main.jsx          # 애플리케이션 진입점
│   └── assets/           # 이미지, 폰트 등 리소스
├── public/               # 정적 파일
└── package.json
```

---

## 🗺️ 라우팅

| 경로     | 페이지   | 설명                                  |
| -------- | -------- | ------------------------------------- |
| `/`      | Home     | 모든 일기를 조회하는 메인 페이지      |
| `/new`   | New      | 새로운 일기를 작성하는 페이지         |
| `/diary` | Diary    | 일기를 상세히 조회하는 페이지         |
| `*`      | Notfound | 존재하지 않는 경로 접근 시 404 페이지 |
